‚ÑπÔ∏è Important

This demo extension requires Firefox Developer Edition to install unsigned extensions. Standard Firefox will not allow installation of unsigned .xpi files.

1

Download Firefox Developer Edition

If you don't already have it, download Firefox Developer Edition from Mozilla:

https://www.mozilla.org/firefox/developer/

2

Enable Unsigned Extensions

Open Firefox Developer Edition and configure it to allow unsigned extensions:

  • Type about:config in the address bar and press Enter
  • Click "Accept the Risk and Continue"
  • Search for: xpinstall.signatures.required
  • Toggle it to false (double-click the value)
3

Download the Extension

Click the download button above to save veritas-v2.0.xpi to your computer. Remember where you save it!

4

Install the Extension

Open the downloaded .xpi file in Firefox Developer Edition:

  • Method A: Drag and drop the .xpi file onto Firefox
  • Method B: Go to about:addons ‚Üí Click the gear icon ‚Üí "Install Add-on From File" ‚Üí Select veritas-v2.0.xpi
5

Confirm Installation

When prompted, click "Add" to confirm installation. You should see the VERITAS icon appear in your browser toolbar.

üîß Troubleshooting

Extension Won't Install

  • Verify you're using Firefox Developer Edition (not regular Firefox)
  • Confirm xpinstall.signatures.required is set to false in about:config
  • Try restarting Firefox after changing the setting

Badge Not Appearing

  • Make sure you're on a supported news website (see list above)
  • Try refreshing the page after installation
  • Check that the extension is enabled in about:addons
