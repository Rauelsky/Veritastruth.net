## ğŸš€ How to Use This Package

### Option 1: Local Viewing (Test Before Upload)

1. **Keep the folder structure intact** - don't move files around
2. **Open** `VERITAS_Interactive_Demo_Web.html` in your web browser
3. **Explore** all 4 topics and 12 articles
4. **Verify** that clicking on articles shows the original sources

### Option 2: Website Deployment (veritastruth.net)

1. **Upload the entire VERITAS_Web_Package folder** to your web server
2. **Maintain the folder structure exactly as-is**
3. **Link to it** from your main website:
   ```html
   <a href="/VERITAS_Web_Package/VERITAS_Interactive_Demo_Web.html">
     View Interactive Demo
   </a>
   ```
4. **Test it** by visiting the link on your live website

### Option 3: Portable Demo (USB Drive, Email)

1. **Zip the entire VERITAS_Web_Package folder**
2. **Share it** via email, USB drive, cloud storage, etc.
3. **Recipients extract and open** `VERITAS_Interactive_Demo_Web.html`
4. **Everything works offline** - no internet required!
